# Nudgen: AI-Powered Email Retention & Automation

This is the core Nudgen skill package. It equips agents with the knowledge and tools needed to build, manage, and optimize email-driven retention and automation workflows.

## Capabilities

This package includes specialized sub-skills for:

- **API Integration**: Server-side request design and implementation.
- **CLI Usage**: Terminal-based automation and data management.
- **Email Best Practices**: Strategic guidance on deliverability, reputation, and lifecycle flows.

## When To Use

Activate this skill when the user is:
- Setting up a new Nudgen integration.
- Troubleshooting existing email campaigns or deliverability issues.
- Looking for advice on retention strategies or automation logic.
- Using the `nudgen` CLI for team management or data exports.

## Getting Started

If you are new to Nudgen, start by exploring the `api` or `cli` skills for technical implementation, or the `email-sending-best-practices` skill for high-level strategy.
